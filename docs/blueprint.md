# **App Name**: Ram Kasuru Portfolio

## Core Features:

- Interactive Hero Section: Dynamic hero section with parallax 3D elements or particle effects created using Three.js. The intro features Ram Kasuru's name, and title.
- Animated About Section: An animated timeline shows Ram Kasuru's education and career journey. It should use smooth fade-in and staggered reveal effects with icons.
- Dynamic Skill Wheel: A dynamic wheel shows Ram Kasuru's skillset with hover effects. These are implemented using Framer Motion for smooth transitions.
- Animated Experience Timeline: An animated vertical timeline details Ram Kasuru's previous positions with animation on scroll, including company logos and hover glow animations.
- Research & Projects Grid: A grid layout highlights Ram Kasuru's research and projects with hover zoom and glass overlay effects and badge support.
- Contact Form with EmailJS: Contact section using a minimal form with fields for Name, Email, and Message. It will use EmailJS for email sending with animated submission button with loading state.

## Style Guidelines:

- Primary color: Saturated dark blue (#3F51B5), complementing a modern look and linking back to artificial intelligence without being too clich√©.
- Background color: Very dark desaturated blue (#0A0D1A), establishing a modern, sophisticated dark scheme.
- Accent color: Bright lavender (#B388FF) for highlighting key elements such as call-to-action buttons and interactive components.
- Font for body and headlines: 'Inter', a sans-serif font providing a clean and modern aesthetic.
- Use minimalistic line icons with subtle hover animations.
- Employ full-width sections, glassmorphism cards, and subtle shadows to create depth and visual interest.
- Use scroll-triggered fade-ins (Framer Motion), parallax 3D elements, and smooth transitions.
